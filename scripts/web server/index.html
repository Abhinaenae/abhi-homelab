Abhi's Homelab

Welcome to my homelab.

Overview

This homelab currently consists of solely an old desktop computer. It is now being used as a lab environment for me to learn hands-on skills in virtualization, containers, and networking while also provding me with useful features.

Features

  • Media Server: Jellyfin hosted on a docker container
  • Landing Page: Localhost website on a K8s pod.
  • Email Automation: Emails me every morning to ensure my network is reachable from it.

Hardware

  • CPU: Intel(R) Core(TM) i3-3240 CPU @ 3.40GHz
  • Storage: 1 TB HDD

Software

  • Operating System: Ubuntu 20.04 LTS
  • Virtualization: Proxmox, Oracle VirtualBox
  • Containers: Docker, Kubernetes
  • Automation: Ansible, Terraform

Future Plans

Here are some of my future plans for expanding the homelab:

  • Self-hosting services, such as Ubooquity, Calibre, and MusicBrainz+Headphones
  • Centralized Storage with NextCloud
